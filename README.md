# gold-trading-strategy
Moving Average Crossover trading strategy on Gold using python
## Features
- Download historical Gold prices using yfinance
- Calculates 20-day and 50-day moving averages
- Backtests strategy performance
- Calculates profit and return perctange
- visualizes portfolio growth

## Technologies
  - Python
  - Pandas
  - Matplotlib
  - yFinance
## Results

  - Buy price: 3373.80
  - Sell price: 4375.50
  - Profit: 1001.70
  - Return: 29.69%
